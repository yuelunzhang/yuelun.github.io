安装操作系统

1、创建虚拟机

2、选择硬件

3、选择操作系统类型

4、配置CPU、内存、硬盘

5、挂在window 2012 R2的ISO镜像

6、安装系统

此处需要设置密码

转换模板之前的工作

1、安装VMware Tools

2、需要选择完整安装（特殊功能支持：NSX等）

安装后，重启

3、操作系统配置

确认制作成模板之前的配置环境

1、开启远程桌面

2、关闭防火墙

3、更新、补丁

4、KMS

5、常用软件等

安装完常用软件后关机即可

关机后，可将该虚机克隆为模板

使用自定义规范从模板新建虚拟机

1、选择虚拟机自定义规范

2、新建自定义规范：

名称 – 描述 – vCenter — 客户机操作系统

所有者名称和组织，根据实际情况填写即可

需要选择“在克隆/部署向导中输入名称”

去掉服务器许可证信息

不设置默认的管理员密码，用户使用虚拟机之时设置自己管理的密码即可

选择时区

此出默认即可

选择手动设置

编辑网卡信息

选择“使用规范时，输入IP等网络信息”

如果环境中有域，请添加，否则工作组即可

从模板部署虚拟机

完成，自此，自定义规范制作完成，我们需要在部署虚拟机的时候应用该规范

设置虚拟机名称

选择存储位置

此处三个都需要勾选

选择我们之前添加的自定义规范

输入主机名、IP地址等

根据实际情况自定义硬件

我们通过虚机 — 监控 – 事件，可以查看到部署后的虚拟机已经使用了自定义规范

进行系统后，管理员设置密码即可使用
